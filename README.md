
# Proyecto de Gestión de Turnos Médicos

Este proyecto es una aplicación web para la gestión de turnos médicos. Incluye roles para administradores y médicos, permite asignar turnos, modificarlos y enviarlos mediante encuestas.

## Estructura del Proyecto
- **Backend**: Express, MongoDB
- **Frontend**: React, Material UI

## Funcionalidades
- **Login** para médicos y administradores.
- Los administradores pueden modificar y eliminar turnos.
